# cyber-seguridad

Instrucciones para abrir la pagina:
  1-Abrir la consola de comandos (cmd) en la carpeta de "cyber-seguridad".
  2-Correr el comando "php -S localhost:8080".
  3-Correr Apache y Mysql via XAMPP.
  4-Abrir la pagina!
